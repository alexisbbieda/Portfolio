CLIENT —

Safe Parking LA

PROJECT —

coming soon

Safe Parking LA is a program that provides parking lots for the homeless to park in at night so they can legally sleep in their cars.


The project presented below is a low fidelity mobile web application that is currently being further tested and developed. The goal of the project is to assist security guards and attendants in their resposibilities and the check-in process.

PROCESS —

We began researching and learning about Safe Parking LA. Once we had a good understanding of the current process we created an application architecture and basic wirerframes. After analyzing the wireframes we created a low fidelity prototype using Figma.

ITERATIONS —

In this early phase of the application we developed a mobile web application using HTML, CSS, JavaScript and Firebase. In this phase we were able to upload a photo and manually fill out the form to gather information about the parking lot resident. After the form is properly filled out by security, the web application searches the database to see if the driver is part of the program. Once that is confirmed they are verified into the parking lot.

SOLUTION —

In the next phase of this project we will be introducing a database that stores the program application requests and uses that to reference for the mobile web applicaiton. We will create a back-end office for their existing website to help automate requests and distribute them amongst the different parking lots.


We will also be updating the usesr experience and flow of the website by conducting more user testing. Once these steps are complete we will create a more resolved and simple interface for the mobile web application.
